Techscape

Mobile game made using Unity (C#)

This project is issued by the Colruyt group, a Belgian supermarket chain, to teach people the different types of mechanical difficulties that take place in stores.

Prototyping

Group work

We started by testing out how to work on mobile games and setting up a few camera setups. This to see from what perspective we would be working for the rest of the project. Furthermore we gathered some reference for puzzle ideas and discussed what we envisioned with the client.

My Contributions

I created some prototypes of how some puzzles could work and how to interact with the puzzles. In the last week of prototyping, I worked on UI giving the player the option to cycle through their inventory and drag items to interact with the scene. Apart from that in cooperation with the other programmers to combine our code and create a demo level for the client(see gif 2).

Production

Group work

The programmers started rewriting code to keep only the most performant and clean for the main project. Meanwhile the artist continued creating assets in the style they agreed with the client.

My Contributions

As I made the UI interactions in the prototype the first task I started on was recreating the inventory. After adding the base functionality of adding and storing objects I made it connect UI to visualize what the player has collected. To top off the inventory functionality I remade option to drag items and slide trough the slots to have a bigger size that's partially off screen.
